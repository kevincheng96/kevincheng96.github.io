Rain Rain Go Away!



A simple weather notification app I developed to teach myself Swift and iOS app development. The app will send a local notification to the user's phone if it is going to rain in the coming hours. Through this project, I learned to use location services, local notifications, API data retrieval, data parsing, and XCode's UI development.

  • Client: Personal project to learn Swift and Xcode
  • Date: Summer 2015
  • Service: App Development

Coco's Adventure



A small and simple "game" I made about my pet dog Coco jumping over moving beanbags. I used Swift's SpriteKit graphics rendering and physics engine to create this game.

  • Client: Personal project to learn simple game development
  • Date: Summer 2015
  • Service: Game Development
