Rain Rain Go Away!



A convenient weather notification app I developed to teach myself Swift and iOS app development. The app will send a local notification to the user's phone if it is going to rain in the coming hours. Through this project, I learned to use location services, local notifications, API data retrieval, data parsing, and XCode's UI development.

Tab Saver



A Google Chrome extension created using HTML/CSS and vanilla Javascript. This extension allows the user flexibility in saving tabs for futuring use and reference. The extension's clean UI allows users to save their current tab, current window, or current session...all with the click of a button.

Happenstance



Happenstance is a hybrid app created using the Ionic framework and Google Firebase backend. Happenstance helps engage users with their surrounding neighborhood. Whenever a user nears a location with a rich and interesting history, the user will receive a notification on his phone. The app can then be opened to display an interesting soundbyte about the history of that location.

MyownGains



A pushup counter app created using Swift and the Myo Armband Swift SDK. The app tracks the number of good form pushups a user completes by processing the gyroscope data stored in a Myo Armband. The counter can be started and stopped through voice recognition (using the OpenEars API).

Facememo



A hybrid contact management app built using the Supersonic framework and Firebase backend. The app displays the faces of professional attendees at professional events to allow users to easily recall and connect with each other.

Hangman



A hangman game that teaches the player a new word (along with its definition) each game. Created with Python and GUI created with Python's tkinter.

Coco's Adventure



A small and simple game I made using Swift in Xcode. The objective is to jump over the moving beanbags using my pet dog Coco. I used Swift's SpriteKit graphics rendering and physics engine to create this game.
